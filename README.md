# listaj.mk — бизнис директориум

Онлајн платформа за **листање и промоција на бизниси во Македонија**. Сопствениците креираат профили; корисниците пребаруваат по категорија, град, име или услуга.

Изградено со [Next.js](https://nextjs.org) 16 (App Router), React 19, TypeScript и Tailwind CSS v4. Корисничкиот интерфејс е на **македонски**; URL-патеките се латински (на пр. `/avto-uslugi`).

## Функционалности

- **Почетна**: пребарување, избрани категории, популарни / нови / најдобро оценети бизниси (демо податоци)
- **Страници на категории** (`/[slug]`): подкатегории, филтри по град, оцена и ниво на цена
- **Сите категории** (`/kategorii`)
- **Пакети** (`/paketi`): бесплатно и премиум листање (399 / 699 / 999 ден)
- **Додај бизнис** (`/dodaj-biznis`): форма со server action (демо, без база)

## Покренување локално

Потребни се Node.js 20+ и npm.

```bash
npm install
npm run dev
```

Отвори [http://localhost:3000](http://localhost:3000).

## Скрипти

| Команда       | Опис                          |
| ------------- | ----------------------------- |
| `npm run dev` | развој (Turbopack)            |
| `npm run build` | продукциска градба          |
| `npm run start` | стартување на production сервер |
| `npm run lint`  | ESLint                       |

## Структура на проектот

- `app/` — страници и layout (`page.tsx`, `[slug]/`, `paketi/`, `dodaj-biznis/`, …)
- `components/` — заеднички UI (`SiteHeader`, `BusinessCard`, …)
- `data/categories.ts` — главни и подкатегории
- `data/businesses.ts` — демо огласи и филтри / пребарување
- `types/business.ts` — типови за бизнис записи

## Следни чекори
